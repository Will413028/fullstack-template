## ADDED Requirements

### Requirement: Complete metadata in root layout
The root layout SHALL include comprehensive metadata for SEO and social sharing.

#### Scenario: OG tags present
- **WHEN** a page is shared on social media
- **THEN** Open Graph title, description, and type SHALL be present

#### Scenario: Sitemap includes all public routes
- **WHEN** search engines fetch the sitemap
- **THEN** all public locale routes SHALL be listed
